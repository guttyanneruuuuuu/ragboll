# RAGBLADE ARENA 設計メモ

このドキュメントは、Ragdoll Blade 由来の爽快感（ラグドール物理・関節ロック・一撃必殺）を維持しつつ、
ブラウザで軽量動作するオンライン対戦型への拡張方針をまとめたもの。

## 実装済みコア
- 3Dラグドール + 剣戟 + 関節ロック
- PWAオフライン対応
- 友達ルームのオンライン対戦（WebRTC）
- ヒット時バイブレーション

## 今回追加
- 武器プリセット（刀 / 大剣 / 槍）
- ステージ選択（通常 / リングアウト / 回転刃）
- ハザード接触ダメージ

## 次のマイルストーン
1. ルーム最大人数を2→4へ拡張
2. 武器固有アクション（突進、パリィ破壊）
3. 低通信量モード（状態同期を低頻度化）
4. GitHub Pages自動デプロイ確認手順のCI化
